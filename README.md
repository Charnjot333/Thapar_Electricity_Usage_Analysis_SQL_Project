# ⚡ Thapar Electricity Usage Analysis SQL Project

A real-world **Database Management System (DBMS)** project developed as part of the **Database Management Systems** course at **Thapar Institute of Engineering and Technology (TIET)**.

This project demonstrates how SQL can be used to organize, manage, and analyze electricity usage data collected from multiple buildings. The objective was to design an efficient relational database and extract meaningful insights using SQL for better monitoring of electricity consumption, billing, and energy management.

---

## 📖 Project Overview

Electricity management is an important aspect of every educational institution and organization. Large campuses generate a significant amount of electricity usage data, making manual record management inefficient and time-consuming.

In this project, electricity usage records collected from different buildings were organized into a relational database using **MySQL**. Various SQL operations were performed to analyze electricity consumption, billing records, meter readings, payment status, complaints, and high electricity usage patterns.

The project highlights how SQL can transform raw operational data into valuable insights that support better decision-making and resource management.

---

## 🎯 Objectives

- Design a normalized relational database for electricity management.
- Store electricity usage records in an organized manner.
- Analyze electricity consumption across different buildings.
- Monitor billing and payment information.
- Track electricity-related complaints.
- Identify buildings with high electricity consumption.
- Generate meaningful reports using SQL queries.
